#### Features / Routes

Model Routes

- GET : `/movies` or `/music`
- GET : `/movies/id` or `/music/id`
- POST : `/movies` or `/music`
- PUT : `/movies/id` or `/music/id`
- DELETE : `/movies/id` or `/music/id`

Auth Routes

- POST : `/signup`
- POST : `/signin`
- GET : `/users`

Other Routes

- GET : `*` - catch-all route
